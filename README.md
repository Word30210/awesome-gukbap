- 규모: 1층, 많은 인원 수용 불가
- 운영 시간
  - 평일: 11:00 ~ 20:00
  - 토: 10:45 ~ 14:30
  - 일: 휴무
- 메뉴
  - 국밥류: 11,000
  - 국밥 정식류: 18,000
- 와이파이 유무: 있음

  - 식당이 작음, 바 테이블 형식 + 식탁 좌석 조금
  - 돼지 국밥인데 잡내 하나도 없고, 국물이 맛 있음

#### [진국 순댓국](https://naver.me/F1MyDQyJ)
- 위치: 서울 관악구 봉천동 28-45
- 규모: 1층, 많은 인원 수용 불가
- 운영 시간
  - 평일, 휴일: 5:00 ~ 22:00
- 메뉴
  - 순댓국밥: 9,000 특 11,000
  - 토종순대: 8,000 특 10,000
- 와이파이 유무: 있음
- 기타 특징:
  - 당연하게도 당면순대가 아님
  - 양파는 자색양파로 서빙됨
  - 깍두기가 매우 잘 익어있으나 배추김치는 겉절이 느낌임
  - 순대가 6개 이상으로 매우 많으며 고기도 많이 들어있음
  - 순대, 머릿고기, 삼겹살 짜투리, 오소리감투 등 정말 많은 종류가 들어있음
  - 순댓국밥의 간이 없다시피해서 간을 세게해도 됨
  - 고추가 인당 한개이지만, 고추의 퀄리티가 매우 좋음
  - 다데기는 매운편은 아니고 2스푼 넣어야 칼칼함이 느껴짐
